Online Cinema Ticket Management System

The Online Cinema Ticket Management System is a Java-based application developed to streamline the process of booking movie tickets. Users can browse available movies, select showtimes, choose seats, and book tickets online. The system also includes admin functionalities for managing movies, show schedules, seat availability, and viewing booking records. Developed using Java (with JDBC and Swing/JavaFX for UI), and backed by a MySQL database, the project ensures smooth user interaction and reliable data management. Key features include user login/registration, real-time seat updates, and a user-friendly interface. You can explore the complete project on my GitHub


Pharmacy Management System

The Pharmacy Management System is a desktop application developed in C# to manage day-to-day operations of a pharmacy efficiently. It allows users to handle medicine inventory, process sales, manage customer records, and generate billing reports. RAILWAY ANTI-COLLISION SYSTEM IN BANGLADESH

The Railway Anti-Collision System is a software engineering project aimed at enhancing train safety by reducing the risk of collisions on railway tracks in Bangladesh. The system uses GPS-based tracking and communication between trains to detect proximity and alert operators in real-time. Developed with a focus on software reliability and system integration, the project includes modules for train tracking, route monitoring, collision prediction, and emergency alert generation. It also features a central control interface for railway authorities to monitor train positions and respond to potential threats. This project highlights how software can play a crucial role in improving public safety within the railway infrastructure of Bangladesh. You can view the full project on my GitHub


Pharmacy Management System

The Pharmacy Management System is a database-driven project developed to streamline the workflow in a pharmacy setting. It manages patient records, medicine inventory, prescriptions, billing, and supplier information efficiently. The system uses SQL (MySQL/Oracle) as its backend database, enabling features like automated stock tracking, medicine expiry alerts, secure prescription handling, and daily sales reports. It includes user roles for pharmacists, administrators, and assistants to ensure role-based access and control. The system ensures data accuracy, secure storage, and quick retrieval for auditing and operational analysis. You can explore the complete project on my GitHub.
